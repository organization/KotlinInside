# HttpRequest

[jvm]\
open class [HttpRequest](index.html)

A fluid interface for making HTTP requests using an
underlying [HttpURLConnection](https://docs.oracle.com/javase/7/docs/api/java/net/HttpURLConnection.html) (or sub-class)
.

Each instance supports making a single request and cannot be reused for further requests.
